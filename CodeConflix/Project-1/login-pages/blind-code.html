BLIND CODE













Rules

○ Open to individuals of all skill levels and backgrounds


○ Participants must register in advance to secure a spot


○ Clearly define the problem statements without revealing any information that could identify the author


○ Participants should submit their code and solutions anonymously


○ Set up a blind review process where the evaluators don't know the identities of the participants


○ Define a fair and transparent scoring system in advance
